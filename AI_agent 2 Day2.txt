   > claude --channels plugin:discord@claude-plugins-official --dangerously-skip-permissions
   인스톨 안되어있다고 나오면,

   /plugin install discord@claude-plugins-official
   나갔다가 위 terminal창 입력한 명령어 다시 실행 (--channels)

5. 1차 교안 7일차 Basic 07 수행
   bot 토큰 저장위해 다음 내용 명령실행

   ex.    ~/.claude/channels/discord/.env 경로에 DISCORD_BOT_TOKEN=[DISCORD_BOT_TOKEN] 저장 후 확인을 위해 파일을 열어줘

6. /mcp들어가봐서 discord가 fail이면 엔터해서 reconnect 할지 묻는 화면의 텍스트 복사해서 클로드에게 붙여넣고 해결해달라고 명령입력
   또는 bun 프로그램 설치되어있는지 확인하고 discord mcp 활용할 수 있도록 path 경로 설정해줘 라고 입력

7. /exit 후 클로드 다시 실행(--channels) 그리고 /mcp들어가서 discord가 disconnect인경우 실행하여 reconnect하면 연결됨

10. 클로드가 패어링 코드 입력하면 몇가지 수정을 하고나면 paired되었다고 함. 디스코드 모임창에서도 패어링 되었다고 나옴 학인할 수 있음
