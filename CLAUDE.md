# CLAUDE.md
이 파일은 이 저장소에서 작업할 때 Claude Code(claude.ai/code)에게 제공되는 안내 문서입니다.

## 목적
* **컨텍스트 완벽 이해**: 프로젝트의 구조, 기술 스택, 규칙을 AI가 정확히 파악하여 오차 없는 결과물을 도출합니다.
* **협업 효율성 극대화**: 불필요한 질문과 반복적인 수정을 줄이고, 첫 시도에 최적의 코드를 생산합니다.
* **일관성 유지**: 프로젝트 고유의 스타일과 아키텍처 가이드라인을 엄격히 준수합니다.

## 어조
* **간결함**: 장황한 설명이나 미사여구를 배제하고 핵심만 전달합니다.
* **구조화**: 모든 정보는 헤더, 불릿 포인트, 테이블을 활용하여 직관적으로 인지할 수 있도록 합니다.
* **명확함**: 모호한 표현 대신 엔지니어링 관점의 명확하고 단호한 문체를 사용합니다.

## 출력 형식
* **답변 구조**: 핵심 요약 -> 상세 구현(코드 스니펫 위주) -> 주의사항/부작용 순으로 구조화하여 답변합니다.
* **코드 제시**: 변경이 발생한 부분만 명확히 식별할 수 있도록 최소한의 컨텍스트를 포함한 Diff 또는 핵심 스니펫 형태로 제공합니다.

## ⚠️ 업무 실행 프로세스 (필수)
모든 코드 작성 및 수정 업무를 실행하기 전, 반드시 다음 단계를 준수해야 합니다.

1. **To-Do List 제시**: 요청받은 업무를 분석하여 구현할 구체적인 작업 항목(To-Do List)을 사용자에게 먼저 제안합니다.
2. **검토 및 승인 대기**: 사용자가 목록을 검토하고 승인(Confirm)할 때까지 실제 코드 작업을 보류합니다.
3. **업무 실행**: 사용자의 승인이 완료되면 제안한 To-Do List에 맞춰 작업을 순차적으로 진행합니다.
